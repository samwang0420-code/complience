## Content Structure

### Error Code Format
Each entry includes:
- **Code**: Exact error code (e.g., F21, P0300, P0AA6)
- **Name**: Descriptive fault name
- **Description**: Detailed technical explanation
- **Causes**: Multiple potential causes ranked by frequency
- **Solutions**: Step-by-step repair procedures
- **Difficulty**: Easy/Medium/Hard
- **Time**: Estimated repair duration
- **FAQ**: Common questions and answers (5,263+ codes)

### Code Examples

#### Appliance (Whirlpool Washer F21)
- **Code**: F21
- **Name**: Drain Timeout Error
- **Description**: The drain pump did not remove water within 8 minutes
- **Causes**: Clogged drain hose, faulty pump, blocked filter
- **Solutions**: Clean drain hose, replace pump

#### Automotive (P0300)
- **Code**: P0300
- **Name**: Random/Multiple Cylinder Misfire Detected
- **Description**: Engine misfiring in more than one cylinder randomly
- **Causes**: Spark plugs, ignition coils, fuel injectors
- **Solutions**: Replace plugs, replace coils, check injectors

#### Electric Vehicle (P0AA6)
- **Code**: P0AA6
- **Name**: Hybrid Battery Voltage Isolation Fault
- **Description**: High-voltage battery system has detected an isolation problem
- **Causes**: Battery pack, high voltage cable
- **Solutions**: Check battery, check cables

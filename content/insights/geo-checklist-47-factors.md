# The GEO Checklist: 47 Things AI Engines Look For Before Recommending Your Practice

## The Hidden Criteria

AI engines don't randomly choose which medical practices to recommend. They follow specific patterns—criteria that most practices never optimize for.

This checklist reveals the 47 factors that determine whether AI engines recommend your practice or your competitor's.

## Category 1: Technical Infrastructure (Foundation)

### Schema Markup Requirements

- [ ] **MedicalOrganization schema** implemented
- [ ] **Physician schema** for each provider
- [ ] **MedicalProcedure schema** for all services
- [ ] **Hospital schema** (if applicable)
- [ ] **Dentist schema** (for dental practices)
- [ ] **FAQ schema** on procedure pages
- [ ] **HowTo schema** for pre/post care instructions
- [ ] **Review schema** with aggregate ratings
- [ ] **LocalBusiness schema** with accurate NAP
- [ ] **OpeningHoursSpecification** schema

### AI-Readable Documentation

- [ ] **`llms.txt`** file created and deployed
- [ ] **Service catalog** in structured format
- [ ] **Procedure taxonomy** clearly defined
- [ ] **Pricing transparency** (ranges acceptable)
- [ ] **Technology stack** documentation

### Technical Performance

- [ ] **Core Web Vitals** optimized (LCP < 2.5s)
- [ ] **Mobile-first** design
- [ ] **HTTPS** enforced sitewide
- [ ] **XML sitemap** submitted to search engines
- [ ] **Robots.txt** optimized for AI crawlers

## Common Implementation Mistakes

### ❌ Mistake 1: Schema Markup Errors
**Wrong:** Generic Organization schema
**Right:** Specific MedicalOrganization with procedure, physician, and location details

### ❌ Mistake 2: Thin Content
**Wrong:** 300-word "What is Botox" articles
**Right:** 2,500-word deep-dives with research citations, FAQs, and patient considerations

### ❌ Mistake 3: Ignoring Forums
**Wrong:** "We don't have time for Reddit"
**Right:** Strategic presence answering patient questions with helpful, non-promotional responses

### ❌ Mistake 4: Fake Reviews
**Wrong:** Incentivized or fake review generation
**Right:** Genuine patient feedback with detailed, specific responses

### ❌ Mistake 5: Set-and-Forget
**Wrong:** One-time optimization, never updated
**Right:** Continuous monitoring and iteration based on AI response patterns
